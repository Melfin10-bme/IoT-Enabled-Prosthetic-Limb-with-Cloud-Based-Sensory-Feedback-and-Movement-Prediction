# IoT-Enabled-Prosthetic-Limb-with-Cloud-Based-Sensory-Feedback-and-Movement-Prediction
IoT-enabled prosthetic limb using ESP32 integrates FSR and EMG sensors with AI for movement prediction and haptic feedback. Features real-time cloud monitoring via Blynk/AWS, low-cost 3D-printed design, BLE/Wi-Fi connectivity, and servo-based actuation for natural, adaptive hand motion with >95% accuracy.
